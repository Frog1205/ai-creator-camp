# AI游戏4日创造营 · PBL-AI项目资料包

> **在线体验** → https://frog1205.github.io/ai-creator-camp/ （部署后访问）

## 项目主题
**驱动问题（Driving Question）**：你能不能在4天内，用AI做出一个让同学"停不下来"的小游戏？

## 项目产出（Deliverables）
1. 一个可在浏览器中运行的Web小游戏（HTML/JS单文件）
2. 一份3分钟路演PPT（含游戏截图、玩法说明、AI协作过程）
3. 一份项目复盘记录（AI对话节选 + 迭代日志）

## 4日营整体节奏

| 日期 | 主题 | 核心任务 | 阶段产出 |
|------|------|----------|----------|
| Day 1 | **启动·探索** | 理解AI能力边界，确定游戏创意 | 游戏创意一页纸 |
| Day 2 | **设计·开发** | 用AI协作生成可运行原型 | v0.1 可玩原型 |
| Day 3 | **测试·迭代** | 玩家测试、加关卡、美化UI | v1.0 完整版本 |
| Day 4 | **路演·答辩** | 展示作品、互评、复盘 | PPT + 复盘 |

## 适用对象
- **学生**：10–16岁，零编程基础也可（依赖AI生成代码）
- **班型**：6–24人，建议2人/组或3人/组（也支持AI探索营V2的"个人独立项目"模式）
- **每日时长**：4–6小时（含休息与午餐）

## PBL-AI教学法核心
本营遵循 PBL（项目式学习）+ AI 协作的双引擎：
1. **真问题驱动**：从"做出一个好玩的游戏"这一真实任务倒推所需技能
2. **AI作为协作者**：学生不是"学AI"，而是"用AI"做事——AI是队友不是老师
3. **公开成果**：第4日的路演是项目终点，也是学习的真正动机
4. **反思可见**：每日复盘记录AI对话，让"看不见的思考"变得可被讨论

## 与 AI探索营（一日营）的差异

| 维度 | AI探索营·一日营 | AI创造营·4日营（本资料） |
|------|----------------|-------------------------|
| 时长 | 1天高密度 | 4天有节奏 |
| 个人/团队 | 个人项目为主 | 2–3人小组 |
| 主题 | 三主题赛道（端午/世界杯/自由） | 三赛道（休闲/动作/知识）|
| 产出 | 一款AI游戏 | 游戏 + PPT + 复盘 |
| 终点 | "AI游戏作品首发礼" | 3分钟路演 + 互评 + 颁奖 |
