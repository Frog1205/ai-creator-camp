## 🎯 任务1：和AI"破冰"（60分钟）

打开AI助手（Claude/DeepSeek/豆包都行），完成下面3个挑战，把AI的回答截图保存：

**挑战A**：让AI用一句话告诉你"贪吃蛇游戏的核心规则"。
**挑战B**：让AI给你写一段HTML代码，在浏览器里显示一个红色的方块。把代码保存成 `test.html`，双击打开看看。
**挑战C**：让AI解释这段代码每一行在做什么。

💡 **小提示**：AI不是搜索引擎，你越说清楚要什么，它越能帮到你。

## 🎯 任务2：好游戏拆解（60分钟）

回忆一个你最喜欢的小游戏（《我的世界》《植物大战僵尸》《合成大西瓜》……），用下面的"游戏五要素"拆解它：

| 要素 | 你的回答 |
|------|----------|
| **目标**：玩家在干什么？ | |
| **规则**：能做什么、不能做什么？ | |
| **反馈**：做对/做错时屏幕会怎么变？ | |
| **挑战**：什么让它变难？ | |
| **上瘾点**：为什么你停不下来？ | |

⚡ **小组讨论**：每人分享1分钟，找出"3个让游戏好玩的共同点"。

## 🎯 任务3：游戏创意头脑风暴（30分钟）

用"**X + Y**"公式快速生成10个游戏点子：

> 把一个你熟悉的游戏 X 和一个奇怪的元素 Y 组合。
> 例：贪吃蛇 + 数学题 = "吃数字才能变长的蛇"
> 例：俄罗斯方块 + 做饭 = "拼食材做菜的方块游戏"

写下你的10个点子，圈出3个你最兴奋的。

---

## 🎯 任务4：用AI验证创意（90分钟）

针对你最兴奋的1个点子，问AI这4个问题（**提示词模板**）：

```
我想做一个游戏：[一句话描述你的创意]。
请帮我分析：
1. 这个游戏的核心玩法循环是什么？
2. 用HTML+JavaScript做，技术难度1–5星是几星？
3. 4天时间一个零基础团队能做出来吗？
4. 如果时间不够，可以砍掉哪些功能保留核心？
```

把AI的回答记录下来。如果AI说"难度5星、做不出来"，**换一个创意或砍功能**，再问一次。
