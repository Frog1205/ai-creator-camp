## 🎯 任务2：第一版代码（90分钟）

**黄金提示词模板**：

```
请用单个 HTML 文件（包含 CSS 和 JavaScript）实现以下功能：
[功能1的描述]

要求：
- 代码写在一个 .html 文件里，双击就能在浏览器打开
- 关键代码加上中文注释
- 用 <canvas> 标签来画游戏
- 不要使用任何外部库（不要 jQuery、不要 React）

请直接给我完整代码。
```

**操作步骤**：
1. 把AI给的代码复制到记事本
2. 另存为 `game.html`（注意：保存类型选"所有文件"，不要 `.txt`）
3. 双击打开，看效果
4. **不对就告诉AI**："运行后画面是黑的，应该显示一个绿色的蛇，请修改。"

⚠️ **千万不要**一次让AI做完所有功能。**做完一个，测试一个，再做下一个**。

## 🎯 任务3：调Bug大作战（90分钟）

代码不工作？别慌。按这个流程问AI：

**Bug提示词模板**：
```
我的代码运行后出现这个问题：
[具体描述：什么按钮按了没反应？什么地方颜色不对？什么时候页面卡住？]

我期望的结果是：
[期望描述]

这是我的完整代码：
[粘贴整段代码]

请告诉我哪里错了，并给我修改后的完整代码。
```

🔑 **三个调试技巧**：
1. **按 F12** 打开浏览器控制台，红色字就是错误，截图发给AI
2. 一次只改一个地方，改完立刻测试
3. 改坏了？让AI回到上一版（所以**每次能跑的版本都另存一份**：`game_v1.html`、`game_v2.html`……）

## 🎯 任务4：加上"灵魂玩法"（90分钟）

到现在你应该有一个"能动但很无聊"的游戏。让它**好玩**起来：

从下面挑1–2个加进去：
- [ ] **加分系统**：右上角实时显示分数
- [ ] **难度递增**：越玩越快/越难
- [ ] **音效**：吃到东西"叮"一声（让AI用 Web Audio API 生成简单音效）
- [ ] **彩蛋**：连续吃5个数字触发特效
- [ ] **皮肤切换**：按S键换颜色
- [ ] **历史最高分**：用 `localStorage` 存下来
