## 🎯 任务1：迭代 + 美化（120分钟）

按"**先功能 后美化**"顺序处理反馈：

**功能层**（先做）：
- 玩家"没看懂规则" → 加开始页/教程页
- 玩家"太难/太容易" → 调难度参数
- 玩家"按了没反应" → 加按钮反馈动画

**美化层**（后做）：
让AI帮你升级视觉。提示词模板：
```
请帮我把这个游戏页面变得更好看：
[粘贴当前完整代码]

具体要求：
1. 用深色背景 + 霓虹色（紫/青/粉）做主色调
2. 主标题用渐变色，加发光效果
3. 按钮加悬停动画
4. 字体用更现代的（比如 'Microsoft YaHei' / 等线）

保留所有现有功能不变，只改样式。给我修改后的完整代码。
```

## 🎯 任务2：关卡 / 难度系统（60分钟）

让游戏有**纵深**——玩家能"打通关"或"刷分"。

选一个做：
- **关卡模式**：5关，每关条件不同（地图大小、敌人速度、目标分数）
- **难度模式**：简单/普通/困难，参数表AI帮你写
- **无尽模式**：分数越高怪物越快，挑战世界纪录

**关卡设计提示词**：
```
我想给我的游戏加关卡系统。当前代码：[粘贴代码]
请帮我加入：
- 5个关卡，分别叫"萌新村"、"小试身手"、"渐入佳境"、"高手过招"、"无尽地狱"
- 每关有不同参数（速度、目标分）
- 通关后显示"恭喜！进入下一关"，并自动加载下一关
给我完整修改后代码 + 5个关卡的参数表。
```

## 🎯 任务3：录制Demo视频（60分钟）

为明天路演准备**30秒短视频**：

📱 **手机录屏方法**（最简单）：
- 苹果：从右上角下滑 → 点录屏按钮，对准电脑屏幕拍
- 安卓：通知栏 → 屏幕录制
- 电脑：Win+G（Windows游戏栏）或 QQ Ctrl+Alt+S

🎬 **录什么**（30秒结构）：
- 0–5秒：游戏标题画面
- 5–20秒：最有趣的玩法片段
- 20–30秒：高分时刻 / 通关画面
