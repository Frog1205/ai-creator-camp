🥇 主力工具详解

① 秒哒 AI · 自然语言造应用神器

核心能力：用一段中文描述，自动生成可运行的网页应用（包括小游戏）。零代码门槛，10–16岁学生10分钟出原型。

时机怎么用
Day2 上午把"创意一页纸"贴进秒哒，10分钟出v0.1原型
Day2 下午在原型基础上追加："加3个关卡""加排行榜"
Day3 上午根据玩家反馈追加："把怪物速度调慢一半"
Day3 下午让秒哒加音效、改配色、做开始页

② GLM 智谱清言 · 全程对话伙伴

通用对话、推理、文档分析、创意发散；免费额度极大、中文体验好，4天密集使用也不会用完。

时机怎么用
Day1 全天创意发散、竞品分析、创意一页纸打磨
Day2 卡壳时秒哒结果不理想时，让GLM分析问题、改写提示词
Day3 反馈整理把玩家反馈贴给GLM分类优先级
Day3 测试GLM 扮演10岁玩家给反馈
Day4 路演写3分钟讲稿、整理PPT大纲、答辩问题预演

🥈 辅助工具详解

③ Codex · 进阶代码助手

适用场景：学生想看代码细节、调算法时；老师现场示范"代码层AI协作"。
4日营定位：不强推全员使用——10岁孩子不需要。老师准备1台演示机配Codex作为示范工具。12+ 高阶学生可作为奖励引入。

④ WorkBuddy · IDE代码辅助

适用场景：愿意打开 VS Code 写代码的进阶学生。
4日营定位：选修工具。默认不教，学生主动问起再讲。Day3 学生想"看懂秒哒生成的代码"时引入。

🎨 资产配套工具（Day3 才引入）

资产类型推荐工具关键技巧
图片素材GLM 内置画图 / 即梦AI免费、中文提示词友好
SVG 矢量图让 GLM/秒哒生成 SVG 代码嵌入HTML不需图床，无跨域
图标库iconfont.cn阿里出品，海量矢量图
音效Web Audio API 代码（GLM 生成）纯JS不需音频文件
BGM 音乐海绵音乐 haimian.com字节出品，国内访问稳
PPT 生成GLM 出大纲 → WPS AI 套模板路演冲刺一次教完
Demo 视频手机录屏 + 剪映自动字幕、加BGM
AI 配音海螺 AI / 剪映朗读不想出镜可用
🔑 关键技巧：让 GLM/秒哒 生成 SVG 代码和 Web Audio 音效代码，嵌入网页—— 学生手里没有图床，也避免跨域 / 路径问题。这一招省下 80% 的麻烦。

⏰ 工具引入节奏（教师视角）

核心原则：不要 Day1 把所有工具都讲完。学生注意力是宝贵资源，用到时才引入。
时机引入工具为什么这时
Day1 上午仅 GLM 对话建立信心，一次学一个
Day1 下午+ xmind 思维导图创意需要结构化
Day2 上午+ 秒哒 AI ← 当天主角学生开始"造游戏"的时刻
Day2 下午+ Codex（老师演示）展示"代码层的AI协作"
Day3 上午+ 即梦 AI / SVG 生成美化阶段才需要
Day3 下午+ Web Audio 音效 / 录屏资产丰富 + 准备视频
Day4 上午+ GLM 写讲稿 + WPS AI 出 PPT路演冲刺一次教完
Day4 下午+ 剪映 + 海螺 AI 配音最终视频包装
