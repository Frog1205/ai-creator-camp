# 路演PPT结构模板（3分钟版 · 7页）

> 原则：**一页一个意思，少字多图**。3分钟讲完不慌。

---

## 第1页：封面（5秒）

```
╔═══════════════════════════════╗
║                               ║
║   《游戏名》大字标题            ║
║                               ║
║   一句话定位（副标题）          ║
║                               ║
║   组名 · 成员姓名               ║
║                               ║
╚═══════════════════════════════╝
```

📌 关键：标题大、副标题清楚说明游戏是什么。

---

## 第2页：钩子（20秒）

> 用一个**问题** / **痛点** / **故事** 开场

示例文案：
- "你有没有玩过那种，明明很无聊但停不下来的游戏？"
- "我们想做一个，**会让你妈妈也想偷玩**的小游戏。"

📌 关键：**不要从"大家好我们是XX组"开始**。

---

## 第3页：游戏一图流（30秒）

> 一张**最好看的游戏截图**，旁边写3个关键词

```
[ 大截图 ]        🎮 核心玩法：________
                  ⚡ 独特卖点：________
                  🎯 目标玩家：________
```

📌 关键：截图清晰、关键词不超过3个词。

---

## 第4页：现场演示（60秒）

> 这一页其实不放在PPT里——你**真的去玩**或**播视频**。

如果做不到现场玩：放30秒Demo视频（自动播放）。

📌 关键：演示**最有趣的30秒**，不是全部玩法。

---

## 第5页：AI协作故事（30秒）

> 一个**具体的瞬间**，不是泛泛而谈

示例文案模板：
```
我们卡了一下午，因为AI总是把蛇头画反方向。

后来我们换了问法：
"请把蛇头的旋转角度和移动方向绑定，
不要用if判断每个方向"

AI一次就改对了。

我们学到：和AI说"你要做什么"，
比说"它现在错在哪"更有效。
```

📌 关键：**讲具体的对话**，不要说"AI很有用"这种空话。

---

## 第6页：数据 & 反馈（20秒）

> 数字说话

```
📊 这4天我们：

• 和AI对话了 ___ 轮
• 改了 ___ 个版本
• 让 ___ 个同学试玩过
• 最高得分纪录：___（保持者：___）

最让我们意外的反馈：
"____________"
```

📌 关键：**有真实数字**，比"我们很努力"有说服力。

---

## 第7页：金句收尾（10秒）

> 一句让评委记住你们组的话

文案参考：
- "我们做的不是游戏，是给4天前那个不敢开始的自己一个答案。"
- "AI写了大部分代码，但游戏的灵魂是我们的。"
- "下一步，我们想把它发给___（具体的人/地方），让更多人玩到。"

最后一行：
```
谢谢！我们是___组，欢迎来玩。
（旁边可放游戏截图或二维码）
```

---

## 🎨 PPT视觉建议

| 元素 | 建议 |
|------|------|
| 字号 | 标题 60+，正文 32+ |
| 颜色 | 深色背景 + 高亮色（与游戏风格一致） |
| 字数 | 每页 < 30 字 |
| 图片 | 至少 5 页有大图 |
| 动画 | 0–1 个，不要花 |
| 字体 | 思源黑体 / 微软雅黑（千万别用宋体） |

---

## ⏱ 3分钟讲稿配速表

| 时间 | 内容 | 大概字数 |
|------|------|---------|
| 0:00–0:20 | 钩子 | 70 字 |
| 0:20–0:50 | 创意介绍 | 100 字 |
| 0:50–1:50 | 演示 + 解说 | 200 字（边玩边说） |
| 1:50–2:20 | AI故事 | 100 字 |
| 2:20–2:50 | 数据反馈 | 80 字 |
| 2:50–3:00 | 金句收尾 | 30 字 |

总字数约 **580–600 字** —— 写出来卡时间。
