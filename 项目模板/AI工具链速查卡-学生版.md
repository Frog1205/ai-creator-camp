## 🔑 万能提示词5件套（背下来）

### ①写新功能
```
我在做[游戏类型]游戏，已经完成了[现状]。
请帮我加一个功能：[新功能描述]
要求：单HTML文件、不用外部库、加中文注释
给我完整代码。
```

### ②修Bug
```
我的代码运行后[现象描述]，但我期望[期望]。
控制台报错：[贴报错]
完整代码：[贴代码]
请告诉我哪里错了，给我修改后的完整代码。
```

### ③美化UI
```
保留所有功能不变，只升级视觉：
风格：[深色霓虹/复古像素/手绘可爱]
[贴代码]
给我修改后的完整代码。
```

### ④AI当玩家
```
请你扮演10岁小朋友第一次玩我的游戏：
[规则描述]
请给我3个真实反馈。
```

### ⑤生成素材
```
请用SVG画一个[描述]，宽40高40。
直接给<svg>代码，我嵌到HTML里。
```

## 🚫 这些坑别踩

❌ 让 AI 一次写完整个游戏 → ✅ 拆成5个小功能，一个一个让它写
❌ 代码不工作只说"不对" → ✅ 截图错误+完整代码+期望
❌ AI说"做不出来" → ✅ 砍功能或换工具问
❌ 改坏了不知道怎么回去 → ✅ 跑通就 game_v1.html / v2.html 另存
❌ Day1把所有工具都试一遍 → ✅ 先把 DeepSeek 玩熟
