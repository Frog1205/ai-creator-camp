## 🔑 万能提示词5件套（背下来）

### ① 让秒哒造游戏
```
做一个网页小游戏：
- 玩法：[2句话核心循环]
- 控制：[键盘/鼠标/触屏]
- 视觉风格：[关键词3个，如"深色霓虹、像素风、可爱"]
- 关键功能：[3-5个具体功能]
请直接发布，给我可访问的网页链接。
```

### ② 让秒哒迭代
```
基于刚才的游戏，请帮我加这些功能：
1. [改动1]
2. [改动2]
其他保持不变，发布更新后的版本。
```

### ③ 让 GLM 修提示词（秒哒结果不理想时）
```
我想做的游戏是：[一句话描述]
我给秒哒的提示词是：[贴提示词]
秒哒生成的结果有这些问题：[问题1/2/3]
请帮我改写一版更精确的提示词。
```

### ④ 让 GLM 当玩家
```
请你扮演10岁小朋友第一次玩我的游戏：
[规则描述/截图]
请给我3个真实反馈：
1. 你一眼想做什么？
2. 玩到第30秒你会觉得什么？
3. 你最不爽的一点是什么？
```

### ⑤ 让 GLM 写路演稿
```
我做了一个游戏：[一句话定位]
核心玩法：[3步循环]
独特卖点：[一句话]
请按这个7页结构写3分钟路演稿：
1.封面 2.钩子 3.游戏介绍 4.演示 5.AI协作故事 6.数据反馈 7.金句收尾
```

## 🚫 这些坑别踩

❌ 让秒哒一次做完所有功能 → ✅ 分3轮迭代，每轮加2–3个功能
❌ 提示词只说"做个好玩的游戏" → ✅ 玩法+控制+风格+功能都说清
❌ 秒哒出错就放弃 → ✅ 让 GLM 改提示词再丢回秒哒
❌ AI 说"做不出来" → ✅ 砍功能或换说法再问一次
❌ Day1 把所有工具都试一遍 → ✅ Day1 只玩 GLM，Day2 才碰秒哒
