## 🛠️ 核心工具栈（4个，全程都在用）

### 1️⃣ 对话式AI助手（**主力中的主力**）

| 工具 | 网址 | 强项 | 推荐度 |
|------|------|------|--------|
| **DeepSeek** | chat.deepseek.com | **写代码最强**、免费、国内秒开 | ⭐⭐⭐⭐⭐ 首推 |
| **Kimi (月之暗面)** | kimi.com | 长文本理解、读PDF文档 | ⭐⭐⭐⭐ |
| **豆包** | doubao.com | 字节出品、对小学生友好、有App | ⭐⭐⭐⭐ |
| **通义千问** | tongyi.com | 阿里出品、代码能力强、有"通义灵码"插件 | ⭐⭐⭐⭐ |
| **智谱清言 GLM** | chatglm.cn | 免费额度大、推理稳 | ⭐⭐⭐ |
| **腾讯元宝** | yuanbao.tencent.com | 微信生态、有联网搜索 | ⭐⭐⭐ |

**4日营推荐配置**：
- 学生主力：**DeepSeek**（写代码）
- 学生辅助：**豆包/通义**（创意阶段对话、读PPT等）
- 老师备用：**Kimi**（备课、长文档分析）

⚠️ **不推荐**：Claude/ChatGPT/Gemini —— 国内访问不便、容易卡课。如学生家有条件可作补充。

### 2️⃣ 代码编辑/运行环境

| 工具 | 用途 | 推荐场景 |

**4日营推荐**：Day1–2 用记事本（避免被工具学习挡住），Day3 开始可介绍 VS Code。

## 🎨 进阶工具链（按环节按需引入）

### Day 1 创意期 · 创意辅助
| 工具 | 用法示例 |
|------|---------|
| **DeepSeek/豆包** | "帮我列20个'校园+解谜'的游戏点子" |
| **腾讯元宝**（联网） | "搜一下最近抖音爆款小游戏有哪些特点" |
| **xmind.cn / processon** | 把头脑风暴可视化为思维导图 |

### Day 2 开发期 · 代码协作
| 工具 | 用法 |
|------|------|
| **DeepSeek** | 主力代码生成 |
| **通义灵码**（VS Code插件） | 边写边补全、零基础学生可不上 |
| **Cursor** | AI优先的IDE，进阶学生可试 |
| **Trae** | 字节的AI IDE，国内访问好 |

**推荐工作流**（学生侧）：
```
浏览器开 DeepSeek 对话 → 复制代码 → 粘贴记事本 → 另存game.html → 双击Chrome打开 → 出错截图 → 回到DeepSeek
```

### Day 3 迭代期 · 美术资产生成

| 资产类型 | 工具 | 备注 |
|---------|------|------|
| **游戏icon/角色** | 即梦AI（jimeng.jianying.com）| 字节出品，免费额度大 |
| **像素风游戏图** | 可灵AI（klingai.com）| 快手出品 |
| **背景插画** | 文心一格（yige.baidu.com）| 百度，中文提示词友好 |
| **SVG图标** | 让 DeepSeek 直接生成 SVG 代码 | 推荐！无需图床 |
| **图标库** | iconfont.cn（阿里巴巴矢量图标库）| 海量免费 |
| **配色方案** | coolors.co / 让AI推荐色板 | |

🔑 **关键技巧**：让AI生成 **SVG代码** 嵌入 HTML，**不要用外链图床**——孩子手里没图床，也避免跨域问题。

提示词示例：
```
帮我用 SVG 画一个像素风格的小飞船，宽40高40，要有引擎尾焰。
直接给我 <svg>...</svg> 标签代码，我要嵌入到 HTML 里。
```

### Day 3 · 音效/音乐生成

| 工具 | 用途 |
|------|------|
| **Suno**（suno.com，国内有镜像）| AI生成完整BGM |
| **海绵音乐**（haimian.com）| 字节出品，国内访问稳 |
| **天工 SkyMusic** | 昆仑万维，免费试 |
| **让 DeepSeek 用 Web Audio API 生成音效** | 不用外部文件，最适合学生 |

提示词示例（最实用）：
```
请帮我用 Web Audio API 在游戏里加这3个音效：
1. 接到星星：清脆的"叮"
2. 撞到陨石：低沉的"咚"
3. 通关：上扬的"叮咚~"
不要外部文件，纯JS生成。
```

### Day 3 · AI玩家测试

让 AI 扮演玩家给你反馈——非常省力的"自动化测试"：
```
请你扮演一个10岁的小朋友，第一次玩我的游戏。
游戏规则：[描述]
游戏截图描述：[文字描述]
请给我3个真实反馈：
1. 你一眼看上去想做什么？
2. 玩30秒后会觉得什么？
3. 你最不爽的一点是什么？
```

### Day 4 路演期 · PPT/视频生成

| 工具 | 用途 | 推荐度 |
|------|------|--------|
| **Kimi PPT** | 输入大纲→自动生成PPT | ⭐⭐⭐⭐⭐ 学生最易上手 |
| **AiPPT.cn** | 国产PPT生成、模板多 | ⭐⭐⭐⭐ |
| **WPS AI** | 直接在WPS里生成 | ⭐⭐⭐⭐ |
| **iSlide** | PPT插件、海量模板 | ⭐⭐⭐ |
| **剪映/CapCut** | Demo视频剪辑+字幕+配音 | ⭐⭐⭐⭐⭐ 首推 |
| **必剪/快影** | 简单视频处理 | ⭐⭐⭐⭐ |
| **海螺AI**（minimax）| AI配音、声音克隆 | ⭐⭐⭐⭐ |

**Day4推荐工作流**：
```
DeepSeek 写演讲稿大纲 → 复制大纲到 Kimi PPT 一键生成
→ WPS 微调 → 剪映剪 30秒 Demo 视频（自动字幕）→ 路演
```

## ⚠️ 常见问题预案

| 问题 | 应对 |
|------|------|
| DeepSeek 卡了/连不上 | 切到豆包或通义，提示词通用 |
| 学生家长担心账号注册 | 用老师准备的公用账号轮换 |
| AI生成的图片版权 | 教学场景内使用，**不商用** |
| AI说"我做不了X" | 换说法："请用替代方案" 或换工具 |
| 学生过度依赖AI | 老师规则：**每天至少有1个功能学生不用AI自己想** |

## 🎓 给老师的工具引入节奏建议

| 时机 | 引入哪个工具 | 为什么这时引入 |
|------|--------------|---------------|
| Day1上午 | **只用 DeepSeek 对话** | 一次只学一个工具，建立信心 |
| Day1下午 | 加 xmind/思维导图 | 创意结构化时引入 |
| Day2上午 | 加 Chrome F12 控制台 | 真正写代码时才有必要 |
| Day2下午 | 加 VS Code（可选） | 代码量大时再升级 |
| Day3上午 | 加 即梦AI/SVG生成 | 美化阶段才需要 |
| Day3下午 | 加 Web Audio音效 / 剪映 | 资产丰富阶段 |
| Day4上午 | 加 Kimi PPT | 路演冲刺，一次教会 |

✨ **核心原则**：**不要在Day1把所有工具都讲完**。学生注意力是宝贵资源，**用到时才引入**，每个工具都有自己的"高光时刻"。
